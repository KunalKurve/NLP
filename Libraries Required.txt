SpaCy:
    Industrial-Strength Natural Language Processing
    spaCy is a free, open-source Python library that provides advanced capabilities to conduct NLP on large volumes of text at high speed. 
    It helps you build models and production applications that can underpin document analysis, chatbot capabilities, and all other forms of text analysis.

TextBlob:
    TextBlob is a Python library for processing textual data. 
    It provides a simple API for diving into common NLP tasks such as part-of-speech tagging, noun phrase extraction, sentiment analysis, classification, and more.
